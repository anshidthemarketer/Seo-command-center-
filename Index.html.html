Hosting Guide
How to deploy this app outside Claude — for free or paid hosting

Step-by-Step: Host Outside Claude

1
Download this HTML file
Click the download button in Claude's output panel to save seo-command-center-v2.html to your computer. It's one single file — no dependencies.
2
Add your API keys
Open the file in a text editor (VS Code, Notepad++). Find the line const KEYS = { and replace the placeholder values with your real keys. Save the file.
3
⚠️ CORS Issue — You need a backend proxy
Browser security blocks direct API calls to xAI/Anthropic/OpenAI from a plain HTML file. You have two options:

Option A (Easy) — Netlify + Edge Function: Free, 5 min setup. Deploy the file to Netlify and add one serverless function that proxies the API call. Hides your key from browser.

Option B (Full control) — Your own server: Node.js/Express backend on any VPS (DigitalOcean $6/mo, or free on Railway.app).
→ netlify.com (free tier)   → railway.app (free tier)
4
Recommended: Netlify Deploy (Easiest Path)
1. Create free account at netlify.com
2. Drag & drop your HTML file onto Netlify's dashboard
3. Go to Functions → create netlify/functions/proxy.js
4. Paste the proxy code from the box below
5. Add your API key in Netlify → Site Settings → Environment Variables as GROK_API_KEY
6. Your app is live at yoursite.netlify.app — shareable with clients
5
Netlify Proxy Function Code
Create file: netlify/functions/proxy.js
exports.handler = async (event) => {
  const { provider, payload } = JSON.parse(event.body);
  const apis = {
    grok:    { url:'https://api.x.ai/v1/chat/completions',    key: process.env.GROK_API_KEY },
    claude:  { url:'https://api.anthropic.com/v1/messages',   key: process.env.CLAUDE_API_KEY },
    openai:  { url:'https://api.openai.com/v1/chat/completions', key: process.env.OPENAI_API_KEY },
    gemini:  { url:'https://generativelanguage.googleapis.com/v1beta/models/gemini-pro:generateContent',
               key: process.env.GEMINI_API_KEY }
  };
  const api = apis[provider];
  if(!api) return { statusCode:400, body:'Unknown provider' };
  const headers = { 'Content-Type':'application/json', 'Authorization':'Bearer '+api.key };
  if(provider==='claude') headers['anthropic-version'] = '2023-06-01';
  const resp = await fetch(api.url, { method:'POST', headers, body: JSON.stringify(payload) });
  const data = await resp.json();
  return { statusCode:200, headers:{'Content-Type':'application/json'}, body: JSON.stringify(data) };
};
6
Point the app to your proxy
In the HTML file, find const PROXY_URL and set it to your Netlify function URL:
const PROXY_URL = 'https://yoursite.netlify.app/.netlify/functions/proxy';

The app will route all LLM calls through your secure proxy — API keys never exposed in browser.
7
Custom domain (optional)
In Netlify → Domain Settings → Add custom domain. Point your DNS to Netlify. Your app becomes seo.yourbrand.com. Free SSL included.

🔑 API Keys & Settings

Keys are stored in your browser's localStorage only — never sent anywhere except the respective API. When hosting externally, move keys to server environment variables (see Hosting Guide).

Grok (xAI) — Recommended
Get key at console.x.ai → API Keys. Model: grok-3-latest. OpenAI-compatible format.
Claude (Anthropic)
Get key at console.anthropic.com → API Keys.
OpenAI (GPT-4o)
Get key at platform.openai.com → API Keys.
Gemini (Google)
Get key at aistudio.google.com → Get API Key.
⚠️ Security Note for External Hosting
When running locally or inside Claude — localStorage is safe. When hosting publicly (Netlify, Vercel, VPS), do not store keys in the HTML. Use the Netlify proxy function in the Hosting Guide — your keys stay on the server, never in the browser.
